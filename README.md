# AnchorHelper

A Celeste mod that allows placing an anchor and recalling the player to it. Includes triggers for adding and deleting anchors.

## Features

- Place an anchor at the player’s current position using a configurable button.
- Recall the player to the anchor instantly.
- Loenn/Ahorn triggers for adding and deleting the AnchorManager in maps.
- Works with Everest Mod Loader.

## Installation

1. Copy the compiled .dll to Mods/ in your Celeste folder.
2. Ensure the Graphics folder is included in the same mod folder.
3. Configure the Anchor/Recall button in Mod Options.

## Controls

- Default binding: C key (can be changed in Mod Options)

## Loenn Triggers

- AddAnchor — adds the AnchorManager to the level.
- DeleteAnchor — removes the AnchorManager completely.
